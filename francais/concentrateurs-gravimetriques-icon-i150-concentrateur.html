iCON i150 Concentrateur

 

Comment ICON œuvres récupération des minéraux fin

Le concentrateur iCON est conçu pour capturer tous les minéraux lourds, y compris or, argent et platinoïdes. Il utilise un champ centrifuge de se concentrer très fines, les minéraux libres qui ne sont pas récupérables en utilisant les techniques traditionnelles de petite échelle et les mineurs artisanaux.

iCON technologie est basée sur le lot de type Falcon concentrateur et conçu par les ingénieurs Falcon même que les concentrateurs de conception pour les plus grandes mines du monde.

En fonctionnement, le matériel est alimenté sous forme de mélange de roche broyée et l'eau dans un bol rotatif qui comprend spéciale rainures fluidisé ou radiers pour capturer les lourds.

Périodiquement, un concentré riche est rincé que doit encore être modernisée pour être transformé en un produit final d'or.

 

  Les caractéristiques qui font l'idéal pour les applications iCON concentrateur à petite échelle sont:
  • Faible poids machine
  • Facilité d'installation
  • Une seule partie mobile
  • Utilisation de pièces d'usure facilement disponibles dans le développement des zones
  • Structure rigide propice à l'exécution à grande vitesse
  • Faible coûts d'acquisition et d'exploitation
  • Un éprouvé, robuste et simple mécanique / électrique plate-forme
  • Efficace nettoyer
  • Les ratios de concentration élevée
